Detailed Description

Create a Reader that fills a GeoGrid with the Location data found in a CSV input stream. Data are expected in format: id, province, population, x_coord, y_coord, latitude, longitude, name The population here is the actual population. It is converted to a population fraction that later gets rescaled with the (fictitious) total population count used during simulation.
