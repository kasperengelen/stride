Detailed Description

template<typename E>
class stride::util::Subject< E >

Template for Subject/Observer (or Publish/Subscribe). Offers flexibility in defining event types and having customized callbacks. Despite the shared_ptrs in the Register/Unregister, the Subject takes no ownership of the observer object and only stores a weak_ptr.
