Detailed Description

A Geographic grid of simulation region contains Locations that in turn contain an index to the ContactPools situated at that Location.

Find contactpools in startRadius (in km) around start and, if none are found, double the radius and search again until the radius gets infinite. May return an empty vector when there are really no pools to be found (empty grid).
